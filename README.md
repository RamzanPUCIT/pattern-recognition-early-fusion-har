# Sensor-based Human Activity Classification using Early Fusion

This project implements Human Activity Recognition (HAR) using multimodal wearable sensor data from the CogAge dataset.

## Project Overview
The project uses an Early Fusion approach on data collected from:
- Smartphone sensors
- Smartwatch sensors
- Smart glasses sensors

Three deep learning models were implemented:
- LSTM
- Simple RNN
- CNN + LSTM

## Experimental Settings
- Subject-Dependent
- Subject-Independent

## Selected Subjects
- Adeel
- Wang

## Best Results
### Subject-Dependent
- LSTM: 81.82%
- RNN: 78.79%
- CNN+LSTM: 69.70%

### Subject-Independent
- LSTM: 13.55%
- RNN: 22.90%
- CNN+LSTM: 22.43%
